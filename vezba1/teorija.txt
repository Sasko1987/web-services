---------------- JavaScript -------------------

Сите овие се high order functions кои се вградени во JavaScript

find - се користи за пребарување во низата и го дава првиот резултат кој го исполнува условот. 

map - се користи за мапирање на низата и ни враќа нова низа

filter - се користи за филтрирање (пребарување на податоците) на низата и ни враќа нова низа. 

reduce - се користи за акумулирање на вредности и крајниот резултат е една вредност. најчесто содржи два параметри accumulator и current value

sort - се користи за сортирање на елементите во низата.

Spread operator in array and in an object 

- spread operatorot се користи за клонирање и спојување на две или повеќе низи или објекти, како и за креирање на нова низа за на постоечката низа да и се додадат дополнителни вредности или елементи.

Destructuring 

- со Destructuring се пристапува до вредностите во низите и објектите и може да се зачувуваат во посебни варијабли


-----------------  Modules -------------------
1. Core - Тоа се модули кои што се вградени во nodejs и може да се користат без да инсталираат, потребно е само да се повикаат

2. Local - локалните модули се модули кои ние самите ги пишуваме. Се што е напишано во nodejs е модули

3. Third party - тоа се модули кои е потрено да се инсталираат. нив ги пребаруваме и инсталираме од npmjs, со користење на npm install (moduleName) командата

----------------WEB --------------------
MVC architecture 

- model, view, controller тоа е архитектурата која се користи при развој на апликација. Моделот ја содржи логиката. View се однесува на корисничкиот interface, тоа што го гледа корисникот, а controllers се одговорни за барањата кои се добиваат од корисникот и давање на одговори на тие барања.

Client server architecture

- Клиентот испраќа барање до серверот кој потоа испраќа барање до датабазата. Кога од датабазата серверот добива одговор, одговорот го испраќа до клиентот. Имаме релација: клиент -> сервер -> база -> сервер -> клиент

HTTP/HTTPs

- е core module во nodejs. HTTP ни овозможува да креираме сервер,да ги добиваме барањата кои ги испраќа корисникот и да даваме одговори на неговите барања


Status codes 1xx - Informational - Client can proceed to send the request body 2xx - Success 3xx - Redirection 4xx - Client error 5xx - Server error
